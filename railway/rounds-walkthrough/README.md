deck (`railway/walkthrough/`, walkthrough.agenticburn.com).

challenge; the Round 1/2/3 states reveal as fragments (advance one click at a time).

Self-contained `index.html` (reveal.js 5.1.0 from CDN). Deploy as a static nginx site on Railway.
